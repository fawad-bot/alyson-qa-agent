Agent Engines

The QA Agent is powered by coordinated open-source engines. Each engine validates a different quality dimension and contributes evidence, findings, and publish readiness signals.

Active

Source Inspector

Reads repositories, package files, route definitions, API contracts, environment requirements, and app metadata.

GitHub API · package.json parser · static file inspection
Active

App Discovery Engine

Crawls the app, detects routes, forms, buttons, auth screens, user journeys, tables, protected areas, and external scripts.

Playwright crawler
Active

Playwright E2E Runner

Executes browser-based end-to-end journeys across Chromium, Firefox, and WebKit.

Playwright
Planned

API Contract Runner

Validates API endpoints, response shapes, status codes, auth behavior, and OpenAPI contracts.

Playwright APIRequestContext · Spectral
Active

Accessibility Runner

Checks accessibility issues such as missing labels, contrast problems, ARIA misuse, and keyboard navigation.

axe-core with Playwright
Planned

Performance Runner

Runs performance and Core Web Vitals checks for important routes.

Lighthouse CI
Planned

Security Runner

Runs baseline security checks, secrets scanning, security headers validation, and basic OWASP tests.

OWASP ZAP · Gitleaks · Semgrep
Active

Test Generator

Turns acceptance criteria into Playwright scenarios and suggested test cases.

AI-assisted test generation
Active

Evidence Collector

Stores screenshots, videos, traces, console logs, network logs, API logs, reports, and generated test files.

Playwright traces · artifact storage mock
Active

Report Writer

Summarizes pass/fail results, findings, severity, recommended fixes, and publish readiness.

AI report synthesis
